This plugin simply adds a link inside the editor to launch a campaign in your Postaga Account with whatever post or page you're editing.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/postaga` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress

Once installed, the plugin will add a link inside the post editor to create a new campaign inside your Postaga account.
